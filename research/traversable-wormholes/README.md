# Traversable Wormhole Research Track

This is the OpenWarp track for true spacetime-shortcut travel.

The goal is not to make a ship locally move faster than light. The goal is to make the path shorter.

```txt
normal space path length: D
wormhole throat path length: L
true shortcut condition: L << D
```

For an interstellar target:

```math
T_{normal}=\frac{D}{c}
```

A wormhole shortcut gives:

```math
T_{wormhole}=\frac{L}{c}
```

If `L << D`, the trip is effectively faster-than-light without locally breaking the light-speed limit.

## Core Method

The theoretically working method is:

```txt
1. Create or obtain two connected microscopic wormhole mouths.
2. Stabilize the throat using negative averaged null energy.
3. Prevent horizons by keeping the redshift function finite.
4. Enlarge or chain throats only if the negative-energy scaling can be solved.
5. Move one mouth to the destination by ordinary sublight transport.
6. Use the throat as a shortcut network.
```

## Important Correction

Quantum entanglement alone does **not** create a usable wormhole wherever we want.

A more serious statement is:

```txt
entanglement may provide a bridge-like quantum connection,
but traversability needs a controlled coupling and negative averaged null energy.
```
