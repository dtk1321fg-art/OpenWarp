# GR-Compatible Testbeds

This folder is for testbeds that follow general relativity directly.

A GR-compatible testbed must use real stress-energy as the source:

```math
G_{\mu\nu} = \frac{8\pi G}{c^4}T_{\mu\nu}
```

## Current Candidate Testbeds

### 1. Positive-Energy Photon Shell

A flat interior, positive-energy shell, curved exterior, and photon-radiation steering.

Status:

```txt
best outside-Casimir candidate
```

### 2. Static Positive-Energy Shell

A simpler version:

```txt
Minkowski interior + Schwarzschild exterior
```

Used to test shell stress-energy and junction conditions.

### 3. Dynamic Radiation Shell

A moving or accelerating shell with photon exhaust / null radiation.

Used to test steering without exotic matter.

## What Counts As Success?

Near-term success is not propulsion.

Success means:

```txt
1. exact metric written down
2. stress-energy tensor calculated
3. energy conditions checked
4. metric perturbation estimated
5. source requirements quantified
```

## What Does Not Count?

Do not claim success from:

- visual analogies
- sci-fi language
- negative-energy wishful thinking
- metric drawings without stress-energy
- stress-energy without energy-condition checks
