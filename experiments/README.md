# Experiments

This folder tracks experimental directions relevant to OpenWarp.

The current goal is not to build a warp drive. The goal is to identify the smallest experiment that could move the project from theory toward real stress-energy engineering.

## Experimental Lanes

### 1. Casimir Systems

Focus:

- Vacuum effects between boundaries
- Geometry-dependent forces
- Possible negative energy-density regions
- Nanotech and MEMS measurement methods

Main question:

> Can Casimir-like systems be shaped and tuned in a way relevant to stress-energy engineering?

### 2. Squeezed States

Focus:

- Quantum optics
- Suppressed vacuum fluctuations
- Sub-vacuum effects
- Precision measurement

Main question:

> Can squeezed states create measurable negative-energy-like behavior relevant to energy-condition studies?

### 3. Nanophotonics and Metamaterials

Focus:

- Engineered optical structures
- Vacuum mode control
- Cavity design
- Field confinement

Main question:

> Can material structure shape vacuum modes enough to matter?

### 4. Precision Interferometry

Focus:

- Phase shifts
- Optical path changes
- Time-delay detection
- Extremely small signal measurement

Main question:

> What instrument could detect the first tiny spacetime-coupling signal?

### 5. Analogue Gravity

Focus:

- Fluids
- Optical media
- Condensed-matter systems
- Simulated horizons and metric-like behavior

Main question:

> Which spacetime-engineering ideas can be tested in analogue systems, even if they are not actual gravity?

## Experiment Proposal Template

Every proposal should include:

1. Title
2. Claim being tested
3. Required equipment
4. Expected signal
5. Possible false positives
6. Measurement method
7. Safety considerations
8. Cost estimate
9. Required expertise
10. Why it matters for OpenWarp

## First Target

The first serious experiment proposal should answer:

> What is the simplest lab system that can produce a tunable negative-energy-like or sub-vacuum effect, and what would it take to measure it?
